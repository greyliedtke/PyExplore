## Variable Naming

- P_ = pressure, kpa
- FP_ = Fluid pressure, psi
- T_ = temperature, C
- F_ = force, lbf
- N_ = speed, krpm
- V_ = vibration, 
- VF_ = volumetric flow liters/min
- CV_ = Control Variable
- ACC_ = acceleration
